ProviderTreeNode

sealed class ProviderTreeNode : Parcelable

This class represents a tree structure of FinancialInstitutionGroupNode objects with children. This eventually leads to a leaf object of type CredentialTypeNode, that contains more detailed Provider data.

The tree will always follow the structure:

FinancialInstitutionGroupNode -> FinancialInstitutionNode -> AccessTypeNode -> CredentialTypeNode
