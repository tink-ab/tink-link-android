ValidationResult

sealed class ValidationResult

Represents the result of a validation of the user input for a field. It can be either one of two types depending upon whether the user input was valid or not. If the user input is valid, the result is ValidationResult.Valid If the user input is invalid, the result is ValidationResult.ValidationError
