TinkLink

@Component([ServiceModule, NetworkModule]) @Singleton abstract class TinkLink

Starting point for using Tink Link in your application. Call TinkLink.create to obtain an instance, then use setUser to add authorization capabilities to that instance. After that you can get further access to specific repositories with the context fetched from getUserContext.
